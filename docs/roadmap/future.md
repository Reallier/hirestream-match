## Phase 2：官网对接（规划中）

**目标：** 与 Nuxt 官网实现 SSO 单点登录

### 待办事项

- [ ] 官网生成 JWT Token
- [ ] Cookie 跨子域共享（.yoursite.com）
- [ ] 未登录用户跳转官网
- [ ] Token 刷新机制
- [ ] 登出同步

### 技术方案

```
官网 (www.yoursite.com)
    ↓ 登录后设置 Cookie
    ↓ domain=.yoursite.com
Match 服务 (match.yoursite.com)
    ↓ 读取共享 Cookie
    ↓ 验证 JWT Token
```
